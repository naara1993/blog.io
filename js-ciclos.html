JAVASCRIPT - CICLOS

¿Qué son y para que se utilizan los ciclos en Javascript?

Una de las principales ventajas de la programación es la posibilidad de crear bucles y repeticiones para tareas específicas, y que no tengamos que realizarlas varias veces de forma manual. Existen muchas formas de realizar bucles, vamos a ver los más basicos, similares en otros lenguajes de programación:

  • For
  • While
  • Do.. while
  • for..in
  • for..of

Ciclo for

Este ciclo se utiliza para realizar una cantidad específicas de iteraciones. La sintaxis es la siguiente:

for (inicialización; condición; incrementador){
// el cuerpo del ciclo, el código que se repite mientras que la condición sea verdadera }

Ciclo while

Este ciclo se utiliza para realizar una cantidad indefinida de iteraciones, la cual finaliza según la condicion que se establezca al inicio de la misma.

La sintaxis es la siguiente:

while (condicional) { //codigo que se ejecuta si la condicion se cumple }

Ciclo Do ..while

Este ciclo es parecido al While pero con la diferencia que el código se ejecutará siempre una primera vez antes de ir hacia la condicion, este ciclo finaliza cuando la condicion que se establezca no se cumpla.

La sintaxis es la siguiente:

Do{ //codigo que se ejecuta }
while (condicional) }

Ciclo For ..in

Un bucle for...in solo itera sobre propiedades enumerables que no son símbolo. Los objetos creados a partir de constructores integrados como Array y Object han heredado propiedades no enumerables de Object.prototype y String.prototype. El bucle iterará sobre todas las propiedades enumerables del objeto en sí y aquellas que el objeto hereda de su cadena de prototipos

La sintaxis es la siguiente:

for (variable in objeto)
{//instrucción}

Ciclo For ..of

La sentencia sentencia for...of ejecuta un bloque de código para cada elemento de un objeto iterable, como lo son: String, Array.

La sintaxis es la siguiente:

for (variable of iterable)
{ instruccion }
